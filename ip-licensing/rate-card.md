| Deliverable Type | Typical Range |

Prices vary based on scope, team size, and complexity. A scoping session (1–2 hours, complimentary for qualified engagements) is required before fixed-price proposals are issued.

## Service Category Descriptions

### AI Agent Design & Architecture ($375/hr)

Design of multi-agent systems from first principles: agent role definition, charter authoring, routing logic design, memory architecture, tool integration planning, human-in-loop protocol design, and observability strategy. Appropriate for greenfield projects or significant architectural refactors.

*Typical deliverables:* Agent charter documents, routing specification, architecture decision records, integration diagram, implementation roadmap.

### Architecture Review ($325/hr)

Independent review of existing AI agent implementations or proposed architectures. Covers: agent role clarity, routing design, failure modes, security and permissions model, observability gaps, scalability considerations.

*Typical deliverables:* Written findings report with prioritized recommendations, optional presentation to engineering leadership.

### Implementation — Development ($275/hr)

Hands-on development work: building agent logic, implementing routing, integrating tools and APIs, writing tests, configuring CI/CD for agent workloads, debugging production issues.

*Typical deliverables:* Working code in client repository, PR-based delivery with documentation.

### Training & Enablement ($250/hr)

Structured training for engineering teams on multi-agent system design and operation. Formats include: workshops (half-day, full-day), hands-on labs, code review sessions, and internal enablement curriculum development.

*Typical deliverables:* Training materials, workshop facilitation, recorded sessions (with client consent).

### Technical Writing & Documentation ($200/hr)

Creation or improvement of technical documentation: agent runbooks, architecture guides, API documentation, onboarding materials, playbooks for common operational scenarios.

*Typical deliverables:* Markdown or Confluence-formatted documentation in client systems.

### Strategic Advisory ($400/hr)

Executive and leadership-level advisory on AI strategy, build-vs-buy decisions, vendor evaluation, organizational design for AI teams, and roadmap prioritization.

*Typical deliverables:* Written advisory memos, presentation materials, facilitated working sessions.

## Getting Started

To begin a new engagement or request a scoping conversation:

1. **Describe your project:** A brief overview of your organization, the challenge you're solving, and your timeline
2. **Scoping session:** 60–90 minute complimentary session to understand requirements and scope the engagement
3. **Proposal:** Written SOW with deliverables, timeline, and pricing within 5 business days of scoping session
4. **Kickoff:** Engagement begins upon signed SOW and receipt of initial payment (50% of fixed-price, or first month's retainer)

All engagements are governed by the Standard Consulting Engagement Terms (see `standard-terms.md`).
